## Backend Development (Node.js)

```zsh
$ npm i nodemon -g
$ npm i express
$ npm i dotenv --save
```

## Install

```zsh
$ npm i
```

## Dev mode & Run

```zsh
$ node --watch index.js
$ nodemon index.js

$ npm run dev
$ npm start
```

## Test

```zsh
$ npm migration
$ npm run test-migration

$ npm run test
$ FILE=./test/ut/widget/list.js npm run test
```

## Tag Version

```zsh
$ npm version 1.0.0-p1
$ git push origin v1.0.0-p1
$ git push
```
